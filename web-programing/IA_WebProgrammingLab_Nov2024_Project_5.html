HTML List Types

Unordered List

An unordered list uses bullet points to list items:

  • Apple
  • Banana
  • Cherry

Ordered List

An ordered list uses numbers or letters to sequence items:

  1. Step 1: Gather materials
  2. Step 2: Assemble components
  3. Step 3: Test functionality

Nested List

A nested list contains sub-items within a list:

  • Fruits
    • Apple
    • Banana
  • Vegetables
    • Carrot
    • Spinach

Description List

A description list is used for terms and their descriptions:

HTML
HyperText Markup Language, the standard language for creating web pages.
CSS
Cascading Style Sheets, used for styling web pages.
JavaScript
A programming language used to create interactive effects within web browsers.
